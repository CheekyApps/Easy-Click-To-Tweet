== Installation ==

1. Upload the Easy Pricing Tables plugin file (`easy-click-to-tweet.zip`) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your dashboard for any post or page, simply click the "blue bird" icon
4. After you click the icon, a pop up will allow you to write the text you'd like to tweet and select your tweet box.
4. For further directions, after installing the plugin, in your Wordpress sidebar, read the more indepth directons on how to use the plugin.
